# How We Work

This page defines how the Nheo Automation Team operates day-to-day.

---

## Communication

- Primary channel: WhatsApp
- Async first — not everything needs a call
- Tag teammates when blocked or needing review

---

## Work Assignment

- All tasks live in Linear
- Lead (Lider) assigns tasks per project and sprint
- Check Linear daily for updates on your issues

---

## Sprints

- Sprint length is flexible — defined per project by the Lead
- Each sprint starts with a kickoff and ends with a brief review

---

## General Rules

- Document as you build, not after
- If you're blocked for more than 2 hours, flag it
- Research before building anything new
